listArguments

private ArrayList listArguments(AslTree AstF,
                                      AslTree args)
Gathers the list of arguments of a function call. It also checks that the arguments are compatible with the parameters. In particular, it checks that the number of parameters is the same and that no expressions are passed as parametres by reference.
Parameters:
AstF - The AST of the callee.
args - The AST of the list of arguments passed by the caller.
Returns:
The list of evaluated arguments.

traceReturn

private void traceReturn(AslTree f,
                         Data result,
                         ArrayList arg_values)
Writes the trace information about the return of a function. The information is the value of the returned value and of the variables passed by reference. It also reports the line number of the return.
Parameters:
f - AST of the function
result - The value of the result
