evaluateBoolean

private Data evaluateBoolean(int type,
                             Data v,
                             AslTree t)
Evaluation of Boolean expressions. This function implements a short-circuit evaluation. The second operand is still a tree and is only evaluated if the value of the expression cannot be determined by the first operand.
Parameters:
type - Type of operator (token).
v - First operand.
t - AST node of the second operand.
Returns:
An Boolean data with the value of the expression.

executeInstruction

private Data executeInstruction(AslTree t)
Executes an instruction. Non-null results are only returned by "return" statements.
Parameters:
t - The AST of the instruction.
Returns:
The data returned by the instruction. The data will be non-null only if a return statement is executed or a block of instructions executing a return.

executeListInstructions

private Data executeListInstructions(AslTree t)
Executes a block of instructions. The block is terminated as soon as an instruction returns a non-null result. Non-null results are only returned by "return" statements.
Parameters:
t - The AST of the block of instructions.
Returns:
The data returned by the instructions (null if no return statement has been executed).

listArguments

private ArrayList listArguments(AslTree AstF,
                                      AslTree args)
Gathers the list of arguments of a function call. It also checks that the arguments are compatible with the parameters. In particular, it checks that the number of parameters is the same and that no expressions are passed as parametres by reference.
Parameters:
AstF - The AST of the callee.
args - The AST of the list of arguments passed by the caller.
Returns:
The list of evaluated arguments.

traceFunctionCall

private void traceFunctionCall(AslTree f,
                               ArrayList arg_values)
Writes trace information of a function call in the trace file. The information is the name of the function, the value of the parameters and the line number where the function call is produced.
Parameters:
f - AST of the function
arg_values - Values of the parameters passed to the function

traceReturn

private void traceReturn(AslTree f,
                         Data result,
                         ArrayList arg_values)
Writes the trace information about the return of a function. The information is the value of the returned value and of the variables passed by reference. It also reports the line number of the return.
Parameters:
f - AST of the function
result - The value of the result
arg_values - The value of the parameters passed to the function
