Class Data

  • java.lang.Object
    • interp.Data
public class Data
extends java.lang.Object
Class to represent data in the interpreter. Each data item has a type and a value. The type can be integer or Boolean. Each operation asserts that the operands have the appropriate types. All the arithmetic and Boolean operations are calculated in-place, i.e., the result is stored in the same data. The type VOID is used to represent void values on function returns.
