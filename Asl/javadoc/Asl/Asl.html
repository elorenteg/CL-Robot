Asl

Class Asl

  • java.lang.Object
    • Asl.Asl
public class Asl
extends java.lang.Object
The class Asl implement the main function of the interpreter. It accepts a set of options to generate the AST in dot format and avoid the execution of the program. To know about the accepted options, run the command Asl -help.
