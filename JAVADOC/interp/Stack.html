getStackTrace

public String getStackTrace(int current_line)
Generates a string with the contents of the stack trace. Each line contains a function name and the line number where the next function is called. Finally, the line number in the current function is written.
Parameters:
current_line - program line executed when this function is called.
Returns:
A string with the contents of the stack trace.

getStackTrace

public String getStackTrace(int current_line,

Generates a string with a summarized contents of the stack trace. Only the first and last items of the stack trace are returned.
Parameters:
current_line - program line executed when this function is called.
nitems - number of function calls returned in the string at the beginning and at the end of the stack.
Returns:
A string with the contents of the stack trace.
